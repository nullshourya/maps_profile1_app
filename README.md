# maps_profile-app

## Overview

This Flutter application demonstrates a clean and modular approach to app development

The application consists of multiple UI screens such as a map-based home screen and a user profile screen. 

## Features
- Displays user profile information
- Displays user current location

## Project Structure
- lib/
- |-main.dart

## Packages Used
- flutter

### Prerequisites
- Flutter SDK
- Android Studio 
- Android Emulator or Physical Device

## Getting Started

### Run the App
```bash
flutter pub get
flutter run


```
